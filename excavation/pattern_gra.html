자세히

벤저민 그레이엄은 시가총액이 순운전자본의 2/3 이하면 좋은 매수 기회로 판단했습니다. 이는 시총/순운전자본 비율이 0.67이하인 기업입니다.
- 매수 시점 : 시가총액 < 순운전자본 2/3 (순운전자본=유동 자산 - 유동 부채)
- 매도 시점 : 매수 후 50% 오르거나, 2년이 지나면 수익률에 관계 없이 매도

아래 메뉴에서 시총/순운전자본 비율이 1.0 미만인 기업은 시가총액이 순운전자본(유동자산-유동부채)에 미달하는 기업입니다.
저가에 팔리는 만큼 재무적으로 위험 수위에 있는 기업도 있는 만큼, 무엇보다 재무 안전성 지표를 꼼꼼히 체크한 후에 종목을 고르는 것이 좋습니다.
그레이엄 만족형은 순이익이 매년 증가하는 성장주를 선호 하는 분들께는 다소 답답한 투자전략이 될 수 있으니, 포트폴리오 운영에 참고하시기 바랍니다.

도움말

  • - PER(Price Earning Ratio, 주가수익배수) = 시가총액 / 순이익
  • - PBR(Price Book value Ratio, 주가순자산배수) = 시가총액 / 자본총계
  • - PSR(Price Sales Ratio, 주가매출액배수) = 시가총액 / 매출액
  • - PCR(Price Cash flow Ratio, 주가영업현금배수) = 시가총액 / 영업현금
  • - ROE(Return On Equity, 자기자본이익률) = 순이익 / 자본총계
  • - ROA(Return On Assets, 총자산이익률) = 순이익 / 자산총계
  • - DY(Dividend Yield ratio, 배당수익률) = (현금배당+주식배당) / 시가총액
  • - 영업이익 = 개별 재무제표 최근 4분기 합산 영업이익
  • - 순이익 = 연결 재무제표 최근 4분기 합산 지배지분 순이익

검색된 종목은 종목추천이 아닌 투자 참고용으로만 활용하시기 바랍니다
