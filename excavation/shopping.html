도움말

  • - PER(Price Earning Ratio, 주가수익배수) = 시가총액 / 순이익
  • - PBR(Price Book value Ratio, 주가순자산배수) = 시가총액 / 자본총계
  • - PSR(Price Sales Ratio, 주가매출액배수) = 시가총액 / 매출액
  • - PCR(Price Cash flow Ratio, 주가영업현금배수) = 시가총액 / 영업현금
  • - ROE(Return On Equity, 자기자본이익률) = 순이익 / 자본총계
  • - ROA(Return On Assets, 총자산이익률) = 순이익 / 자산총계
  • - DY(Dividend Yield ratio, 배당수익률) = (현금배당+주식배당) / 시가총액
  • - 영업이익 = 개별 재무제표 최근 4분기 합산 영업이익
  • - 순이익 = 연결 재무제표 최근 4분기 합산 지배지분 순이익

검색된 종목은 종목추천이 아닌 투자 참고용으로만 활용하시기 바랍니다
